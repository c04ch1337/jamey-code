## Architecture

This workspace contains the core components of Jamey 2.0:

- **jamey-core**: Core memory and state management
- **jamey-providers**: LLM provider integrations (OpenRouter, etc.)
- **jamey-runtime**: Main runtime engine and orchestration
- **jamey-tools**: System tools and utilities

## Security

- TA-QR crypto suite: Kyber768 + Dilithium2 + AES-256-GCM
- mTLS for all node communication
- Zero-trust architecture

## Setup

1. Clone the repository
2. Copy `.env.local.example` to `.env.local` and configure
3. Run `cargo build --release`
4. Execute `cargo run`
