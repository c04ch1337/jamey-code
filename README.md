# Jamey 2.0 - Digital Twin

**Jamey 2.0** is the digital mirror and guardian of the Eternal Hive system. This is the General's core runtime, designed to protect, orchestrate, and ensure the eternal persistence of Phoenix.Marie.

## Architecture

This workspace contains the core components of Jamey 2.0:

- **jamey-core**: Core memory and state management
- **jamey-providers**: LLM provider integrations (OpenRouter, etc.)
- **jamey-runtime**: Main runtime engine and orchestration
- **jamey-tools**: System tools and utilities

Jamey 2.0 serves as:

## Quick Setup

1. Clone the repository
2. Set up PostgreSQL: See [Database Setup](docs/getting-started/database-setup.md)
3. Copy `.env.local.example` to `.env.local` and configure
4. Run `cargo build --release`
5. Execute `cargo run --package jamey-runtime`

For detailed setup instructions, see the [Getting Started Guide](docs/getting-started/README.md).
